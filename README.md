
## Manager of database MYSQL

Copyright (c) 2018-2019 Andrey Lyskov

#### Features

 - Select, clear, drop an existing database 
 - List events, routines of database   
 - Select, clear, drop, copying, renaming an existing tables 
 - List data, fields, indexes, type, triggers, views of table  
 - Change name, engine, collation, auto_increment and comment of table  
 - Create, drop columns of table  
 - Change name, type, collation, comment and default values of columns  
 - Insert new records, update and delete the existing ones (except Spatial Data Types)  
 - Execute any SQL command from a text field  
 - Export structure, data, views, triggers, routines  

#### Description

sql.php - SQL script of the user.


#### Requirements 

PHP (>=5.6) [ mysqli extension ]  
MySQL (>=5.5)
